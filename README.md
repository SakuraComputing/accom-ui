# Accomodations UI

React JS / WepPack / JSON / Redux / Jest

## Task

Booking.com UI POC rendering JSON files using Redux.

```aidl
yarn install
```

## Testing

To run the unit tests run the following. (not full coverage as this is a prototype)

```aidl
yarn run test
```

## Application Start

```aidl
yarn run dev-server

localhost:8080
```
